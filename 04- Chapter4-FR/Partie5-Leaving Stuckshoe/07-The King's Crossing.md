# Traverser la chaussée
Le voyage à travers la Traversée du Roi prendra presque une journée entière ; vous trouverez ci-dessous deux rencontres pour cette période. A l'extrémité de la chaussée se trouve le Camp du Roi, où les joueurs se reposeront probablement. Si vos joueurs aiment se battre, vous pouvez leur lancer quelques autres NSRE mortels.
