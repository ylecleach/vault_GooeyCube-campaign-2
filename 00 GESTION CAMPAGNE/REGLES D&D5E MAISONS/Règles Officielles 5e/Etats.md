| État                             | Effets principaux                                                                                                                                                                                      |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| À terre (Prone)                  | Se déplace en rampant ; désavantage aux attaques ; attaques contre elle ont avantage si à 1,50 m ou moins, sinon désavantage.                                                                          |
| Agrippé (Grappled)               | Vitesse réduite à 0 ; aucun bonus de vitesse ; prend fin si l’agrippeur est incapable d’agir ou hors de portée.                                                                                        |
| Assourdi (Deafened)              | N’entend pas ; rate automatiquement les tests nécessitant l’ouïe.                                                                                                                                      |
| Aveuglé (Blinded)                | Ne voit pas ; rate les tests nécessitant la vue ; attaques contre elle ont avantage ; ses attaques ont désavantage.                                                                                    |
| Charmé (Charmed)                 | Ne peut attaquer ni cibler le charmeur avec des effets nuisibles ; le charmeur a avantage aux interactions sociales.                                                                                   |
| Effrayé (Frightened)             | Désavantage aux attaques et tests tant que la source de peur est visible ; ne peut s’en approcher volontairement.                                                                                      |
| Empoisonné (Poisoned)            | Désavantage aux attaques et aux tests de caractéristique.                                                                                                                                              |
| Entravé (Restrained)             | Vitesse réduite à 0 ; attaques contre elle ont avantage ; ses attaques ont désavantage ; désavantage aux sauvegardes de Dextérité.                                                                     |
| Étourdi (Stunned)                | Incapable d’agir ; ne peut bouger ; rate les sauvegardes de Force et Dextérité ; attaques contre elle ont avantage.                                                                                    |
| Incapable d’agir (Incapacitated) | Ne peut effectuer aucune action ni réaction.                                                                                                                                                           |
| Inconscient (Unconscious)        | Incapable d’agir ; tombe à terre ; lâche ses objets ; rate les sauvegardes de Force et Dextérité ; attaques contre elle ont avantage et sont critiques à 1,50 m.                                       |
| Invisible                        | Ne peut être vue sans magie ou sens spécial ; considérée cachée ; attaques contre elle ont désavantage ; ses attaques ont avantage.                                                                    |
| Paralysé (Paralyzed)             | Incapable d’agir ; ne peut bouger ni parler ; rate les sauvegardes de Force et Dextérité ; attaques contre elle ont avantage et sont critiques à 1,50 m.                                               |
| Pétrifié (Petrified)             | Transformée en matière solide (souvent pierre) ; poids ×10 ; vieillissement suspendu ; incapable d’agir ; résistante à tous les dégâts ; immunisée au poison et maladies (suspendus si déjà présents). |
| Épuisement (Exhaustion)          | État spécial à 6 niveaux : 1. Désavantage aux tests ; 2. Vitesse réduite de moitié ; 3. Désavantage aux attaques et sauvegardes ; 4. PV max réduit de moitié ; 5. Vitesse = 0 ; 6. Mort.               |
